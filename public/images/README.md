# Images Directory

Place image assets here for the Nature Explorer app.

## Recommended Images

### Icons
- `icon-192x192.png` - App icon for mobile home screen
- `icon-512x512.png` - Large app icon
- `favicon.ico` - Browser tab icon

### Backgrounds (Optional)
- `forest-bg.jpg` - Forest scene background
- `ocean-bg.jpg` - Ocean biome background
- `desert-bg.jpg` - Desert biome background

### Creatures (Optional - Currently using emojis)
If you want custom illustrations instead of emojis:
- `bird.png`

- `bat.png`
- `firefly.png`

### Plants (Optional - Currently using emojis)
- `tree-stage-*.png` (stages 0-4)
- `flower-stage-*.png` (stages 0-4)
- `bush-stage-*.png` (stages 0-4)

## Image Specifications

### Format
- PNG with transparency for creatures/plants
- JPG for backgrounds
- SVG for icons (scalable)

### Size
- Icons: 192x192, 512x512
- Creatures: 200x200 to 400x400
- Backgrounds: 1920x1080 or larger
- Optimize all images (use TinyPNG, ImageOptim)

### Style Guidelines
- Soft, rounded shapes
- Bright, vibrant colors
- Friendly expressions
- No sharp edges
- Consistent art style
- Safe for children

## Current Implementation

The app currently uses emoji for all visual elements, which:
- ✅ Loads instantly
- ✅ Works on all devices
- ✅ Consistent across platforms
- ✅ Zero download size
- ✅ Unicode standard

Custom images are optional enhancements!

## Future: Custom Illustrations

If creating custom art, follow the **ART PROMPT**:

"Create cute, vibrant, vector-style illustrations for a kids' nature exploration app. 
Include smiling animals (birds, rabbits, deer), playful clouds, blossoming plants, 
magical insects, soft shadows, rounded shapes, bright pastel colors, and gentle textures. 
Style should feel like a mix of Cocomelon + Duolingo Kids + a picture book. 
Avoid sharp edges. Make all creatures friendly, expressive, and interactive-looking."
